# Регистры форматов файлов

> **Примечание**: Основная статья о регистрах форматов файлов находится в разделе [Форматы файлов](/kb/instruments/file-formats/format-registries).

Регистры форматов файлов предоставляют информацию о различных форматах файлов, их характеристиках и рисках сохранения.

## Основная статья

Для полной информации о регистрах форматов файлов, их использовании и рекомендациях, см.:

**[Реестры форматов файлов](/kb/instruments/file-formats/format-registries)**

Эта статья включает:
- Подробное описание всех основных регистров (PRONOM, LC FDD, Wikidata, Digipres.org и др.)
- Сравнение регистров
- Рекомендации по использованию
- Best practices для работы с регистрами
- Рекомендации для Ruarxive
