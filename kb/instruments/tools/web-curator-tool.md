# Web Curator Tool

**Web Curator Tool** — это open-source система управления workflow для селективной веб-архивации.

## Описание

Web Curator Tool разработан для библиотек и архивов, которым нужен полный контроль над процессом архивации с workflow управлением.

### Особенности

*   **Workflow управление**: Полный цикл от номинации до архивации
*   **Селективная архивация**: Выбор конкретных ресурсов для архивации
*   **Роли пользователей**: Разные роли с разными правами
*   **Интеграция с Heritrix**: Использует Heritrix для фактической архивации

## Установка

### Требования

*   Java 8+
*   Tomcat 8+
*   MySQL или PostgreSQL
*   Heritrix

### Установка

1. Скачайте последнюю версию с [GitHub](https://github.com/DIA-NZ/webcurator)
2. Установите зависимости
3. Настройте базу данных
4. Разверните на Tomcat

## Использование

### Workflow

1. **Номинация**: Пользователи номинируют ресурсы для архивации
2. **Одобрение**: Кураторы одобряют номинации
3. **Архивация**: Система запускает Heritrix для архивации
4. **Валидация**: Проверка качества архивов
5. **Публикация**: Публикация архивов

### Роли

*   **Номинатор**: Может номинировать ресурсы
*   **Куратор**: Может одобрять и управлять заданиями
*   **Администратор**: Полный доступ к системе

## Конфигурация

### Настройка Heritrix

```xml
<!-- webcurator.xml -->
<heritrix>
  <host>localhost</host>
  <port>8443</port>
  <username>admin</username>
  <password>admin</password>
</heritrix>
```

### Настройка базы данных

```properties
# database.properties
db.driver=com.mysql.jdbc.Driver
db.url=jdbc:mysql://localhost:3306/webcurator
db.username=webcurator
db.password=password
```

## Best practices

### Управление номинациями

*   Установите критерии для номинаций
*   Регулярно проверяйте очередь номинаций
*   Обучите номинаторов правильным практикам

### Контроль качества

*   Валидируйте все архивы перед публикацией
*   Проверяйте полноту архивации
*   Документируйте проблемы

### Workflow оптимизация

*   Настройте автоматические процессы где возможно
*   Оптимизируйте ручные шаги
*   Используйте шаблоны для повторяющихся задач

## Сравнение

| Инструмент | Workflow | Роли | Интеграция | Рекомендация |
| :--- | :--- | :--- | :--- | :--- |
| **Web Curator Tool** | ✅ Полный | ✅ | Heritrix | **Для библиотек/архивов** |
| **Browsertrix Cloud** | ⚠️ Ограниченный | ⚠️ | Собственный | Для организаций |
| **Heritrix** | ❌ | ❌ | Нет | Для технических пользователей |

### Когда использовать Web Curator Tool

*   Библиотеки и архивы
*   Нужен полный workflow контроль
*   Селективная архивация
*   Управление ролями пользователей

### Когда использовать другие инструменты

*   Простые проекты (используйте Browsertrix)
*   Автоматическая архивация (используйте Heritrix напрямую)
*   Облачный сервис (используйте Browsertrix Cloud)
