# Инструменты воспроизведения

Инструменты для воспроизведения веб-архивов позволяют просматривать заархивированные веб-страницы так, как они выглядели в момент архивации.

## Обзор инструментов

### OpenWayback

[OpenWayback](openwayback) — классическая реализация Wayback Machine, используемая многими веб-архивами по всему миру.

### PYWB

[PYWB](pywb) — современная Python 3 реализация инструментов воспроизведения веб-архивов.

### ReplayWeb.page

[ReplayWeb.page](replayweb-page) — браузерный, полностью клиентский движок воспроизведения для локальных и удалённых WARC & WACZ файлов.

### InterPlanetary Wayback (ipwb)

[ipwb](ipwb) — воспроизведение веб-архивов с использованием IPFS для распределённого хранения.

### warc2html

[warc2html](warc2html) — конвертация WARC файлов в статический HTML для офлайн-просмотра.

## Выбор инструмента

| Инструмент | Требует сервер | Распределённость | Простота | Рекомендация |
| :--- | :--- | :--- | :--- | :--- |
| **ReplayWeb.page** | Нет | Нет | Высокая | **Для локального просмотра** |
| **PYWB** | Да | Нет | Средняя | **Для веб-сервисов** |
| **OpenWayback** | Да | Нет | Низкая | Для классических установок |
| **ipwb** | Да | Да (IPFS) | Средняя | Для распределённых архивов |
