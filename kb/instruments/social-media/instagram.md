# Instagram

Архивация Instagram сложна из-за агрессивных анти-бот мер платформы.

## Instaloader

[Instaloader](https://instaloader.github.io/) — мощный инструмент командной строки на Python для скачивания фотографий, видео и метаданных из Instagram.

### Установка

```bash
pip install instaloader
```

### Использование

```bash
instaloader profile_name
```

### Важные советы

1.  **Логин**: Для скачивания большинства контента требуется вход в аккаунт (`--login`).
2.  **Лимиты**: Instagram быстро банит за слишком частые запросы. Не запускайте массовую архивацию с основного аккаунта.
3.  **Сессии**: Сохраняйте файл сессии, чтобы не логиниться каждый раз (это подозрительно для Instagram).

## Другие методы

Для создания достоверной копии внешнего вида страниц (включая комментарии) лучше использовать [Browsertricks](../tools/browsertricks) в режиме записи с профилем браузера.
