# ydiskarc (Yandex.Disk)

**ydiskarc** — простая утилита для скачивания публичных папок Яндекс.Диска.

Обычно, чтобы скачать папку целиком, Яндекс требует сохранения её на свой Диск (личное пространство). `ydiskarc` обходит это ограничение и скачивает содержимое публичной ссылки прямо на ваш компьютер.

## Установка

```bash
pip install ydiskarc
```

## Использование

```bash
ydiskarc https://disk.yandex.ru/d/AbCDeFgHiJkLm --output ./my-folder
```

## Особенности

*   Поддерживает вложенные папки.
*   Качает каждый файл отдельно (не требует создания ZIP архива на стороне сервера).
*   Полезно для архивации публичных датасетов или коллекций документов.
