# Инструменты Ruarxive

Инструменты, разработанные командой Ruarxive для архивации различных типов цифровых ресурсов.

## Инструменты

- [tgarc](/kb/instruments/ruarxive-tools/tgarc) - Потоковая архивация Telegram каналов и чатов
- [wparc](/kb/instruments/ruarxive-tools/wparc) - Архивация WordPress сайтов через API
- [ydiskarc](/kb/instruments/ruarxive-tools/ydiskarc) - Скачивание публичных папок Яндекс.Диска
- [filegetter](/kb/instruments/ruarxive-tools/filegetter) - Сбор файлов из публичных источников по URL-паттернам

## Установка

Все инструменты доступны через pip:

```bash
pip install tgarc wparc ydiskarc filegetter
```
