# Yandex Data

Яндекс предоставляет возможность выгрузки данных через сервис управления аккаунтом (Yandex ID). В отличие от Google Takeout, здесь нет единого "центра выгрузки" для всего сразу, но есть инструменты для ключевых сервисов.

Адрес управления данными: [passport.yandex.ru/profile/data](https://passport.yandex.ru/profile/data)

## Яндекс.Диск

Для полного скачивания Яндекс.Диска проще всего использовать:
1.  **Клиент Яндекс.Диска для ПК**: Синхронизировать всё на локальный диск.
2.  **WebDAV**: Подключить диск как сетевой и скопировать файлы.

## Яндекс.Почта

Официального экспорта в MBOX нет. Рекомендуется использовать почтовый клиент (Thunderbird, Outlook) по протоколу IMAP для выкачивания всех писем на локальный компьютер.

## Другие сервисы

На странице "Управление данными" можно запросить архив с информацией из сервисов:
*   Музыка
*   Карты
*   Кинопоиск
*   Маркет

Запрос архива может занять несколько дней. Вы получите уведомление на почту. Данные обычно приходят в формате JSON или CSV.
