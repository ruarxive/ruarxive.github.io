# Курс по цифровой архивации

Курс по цифровой архивации представляет собой серию лекций, посвященных основам сохранения цифрового наследия, веб-архивации и работе с цифровыми архивами.

## О курсе

Курс охватывает основные аспекты цифровой архивации:

- Введение в концепции цифрового сохранения
- Веб-архивация и методы сохранения веб-контента
- Архивация специализированных цифровых ресурсов
- Работа с интернет-архивами

## Лекции курса

### Лекция 1: Введение в цифровую архивацию

[DH.1. Введение в цифровую архивацию](/kb/course/dh1-introduction)

Основы цифровой архивации, понятия born-digital, цели и задачи цифрового сохранения, виды цифровых объектов, форматы файлов и этапы архивации.

### Лекция 2: Веб-архивация

[DH.2. Веб-архивация](/kb/course/dh2-web-archiving)

Методы и инструменты веб-архивации, форматы веб-архивов, практические подходы к сохранению веб-контента.

### Лекция 3: Архивация специализированных ресурсов

[DH.3. Архивация специализированных ресурсов](/kb/course/dh3-specialized-resources)

Архивация различных типов цифровых ресурсов: социальные сети, базы данных, специализированные форматы.

### Лекция 4: Интернет архив с точки зрения цифрового архивиста

[DH.4. Интернет архив с точки зрения цифрового архивиста](/kb/course/dh4-internet-archive)

Работа с интернет-архивами, использование архивов для исследований, практические аспекты работы архивиста.
