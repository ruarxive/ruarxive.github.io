Slack

Возможности экспорта из Slack зависят от вашего тарифного плана и прав доступа (требуется роль Workspace Owner/Admin).

Экспорт данных (Free / Pro)​

  1. Перейдите на страницу Workspace Settings -> Import/Export Data.
  2. Выберите вкладку Export.
  3. Выберите диапазон дат.
  4. Нажмите Start Export.

Что включено:​

  • История сообщений из публичных каналов (JSON).
  • Ссылки на файлы (сами файлы не скачиваются в архиве, их нужно качать скриптами!).

[!WARNING] На бесплатных и Pro тарифах экспорт не включает личные сообщения (DMs) и закрытые каналы.

Корпоративный экспорт (Business+ / Enterprise Grid)​

На тарифах Business+ можно запросить "Corporate Export", который включает личные сообщения и закрытые каналы. Для этого нужно подать заявку в поддержку Slack и подтвердить юридическую необходимость.

Инструменты для скачивания файлов​

Так как стандартный экспорт содержит только ссылки на файлы, для полного бэкапа используйте сторонние инструменты, например slack-export-viewer или специализированные скрипты на Python/Go, которые парсят JSON-экспорт и выкачивают вложенные файлы (attachments).
